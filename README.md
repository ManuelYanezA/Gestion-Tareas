Proyecto personal, un gestor de tareas hecho en Python, para usar tanto en la vida laboral como la vida cotidiana.

#Pestañas
Se pueden agrupar tareas bajo un contexto específico agrupándolas en una pestaña con un nombre en particular (Trabajo, Escuela, Tareas del hogar, etc.)

#Sub tareas
Se puede especificar las tareas en partes más pequeñas, en caso de ser necesario.
